El Modal de **Anami UI** sigue una estética "Square & Clean", utilizando bordes definidos, sombras profundas y un fondo desenfocado para mantener el enfoque en la tarea actual.

## Estructura Base

Un modal se compone de tres secciones principales: Header, Body y Footer.

<Showcase
  htmlCode={`
<div style="background: #f4f4f4; padding: 3rem; border-radius: 8px;">
  <div class="an-modal" style="position: relative; margin: 0 auto; animation: none;">
    <header class="an-modal__header">
      <h3 class="an-text--h3 an-m-0">Título del Modal</h3>
      <button class="an-button">✕</button>
    </header>
    
    <div class="an-modal__body">
      <p class="an-text--body">Este es el cuerpo del modal donde se coloca el contenido principal o formularios.</p>
    </div>
    
    <footer class="an-modal__footer">
      <button class="an-button an-button--outline">Cancelar</button>
      <button class="an-button an-button--primary an-mt-0">Aceptar</button>
    </footer>
  </div>
</div>
`}
/>

## Referencia de Clases BEM

| Clase               | Descripción                                                |
| :------------------ | :--------------------------------------------------------- |
| `.an-modal-overlay` | Contenedor oscuro (`backdrop`) que cubre toda la pantalla. |
| `.an-modal`         | El contenedor principal blanco con sombra.                 |
| `.an-modal__header` | Cabecera con fondo `brand-blob` y borde inferior.          |
| `.an-modal__body`   | Espacio acolchado para el contenido.                       |
| `.an-modal__footer` | Zona inferior alineada a la derecha para acciones.         |

## Variables de Control

El modal utiliza la variable `$z-modal` para asegurar que siempre aparezca por encima de cualquier otro elemento (como el Sidebar o las tablas).

```scss
// abstracts/_variables.scss
$z-modal: 1000;
```
